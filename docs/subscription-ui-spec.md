# Echelon Subscription UI Spec

## Goals
- Make tiers easy to understand.
- Show real quota numbers.
- Show what node contribution earns.
- Make upgrades feel like moving into better service, not just paying a toll.

## Page Sections

### Hero
- Title: "Choose your network tier"
- Subtitle: "Browse, build, and route through the mesh"
- Small trust note: "Free users can still participate as nodes"

### Tier Cards
Each card should show:
- price
- best-for label
- included eepsites
- bandwidth
- AI access
- outproxy access
- routing quality
- button: Select / Current Plan

### Comparison Table
Show side-by-side:
- browser limits
- hosting limits
- AI limits
- privacy features
- relay bonuses
- node reputation bonuses

### Quota Meter Panel
Show current usage:
- page views used today
- bandwidth used today
- eepsite storage used
- AI tokens used
- relay credits earned

### Node Contribution Panel
Show:
- relay uptime
- traffic relayed
- estimated contribution score
- bonus quota unlocked
- next reward threshold

### Upgrade CTA
If over a free limit:
- show upgrade prompt
- explain what unlocks immediately
- show RTD future note as roadmap, not current requirement

## Visual Style
- dark glassmorphism
- strong hierarchy
- bold numbers
- green for positive / earned
- purple or teal for premium accents
- concise copy

## UI Rules
- Do not bury limits in tiny text.
- Every plan must clearly explain what the user gets.
- Free should feel respectable, not crippled.
- Paid should feel meaningfully better.
